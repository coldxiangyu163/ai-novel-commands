# 全书节奏总监

**在规划完多个卷后使用（建议每3卷运行一次），检验全书级节奏健康度。**

> **核心理念**：
> - 长篇失控的根源：plan-volume只管单卷，缺少跨卷节奏规划
> - 单卷节奏好，全书节奏可能崩（如：连续3卷高潮→疲劳）
> - 节奏总监是"全书级体检"，检测拖沓、疲劳、失控风险

---

统计已规划的卷数和总章节数。

## 第二步：提取节奏数据

### 2.1 从各卷大纲提取张力数据

为每一章提取：

- **冲突维度**：单维/二维/三维
- **爽点/爆点**：有/无
- **节奏定位**：开局/升温/高潮/缓冲/收尾

## 注意事项

1. **至少3卷才有意义**：单卷看不出跨卷节奏问题
2. **动态调整**：发现问题后调整大纲，再次运行验证
3. **与其他工具协同**：配合节拍表、爆点曲线、冲突地图使用
4. **不要过度优化**：允许合理波动，不要强求每卷一模一样
5. **后期更重要**：后期卷（卷8-10）的节奏必须强于前期

## 附录：节奏健康标准

### A. 健康的节奏曲线特征

**1. 有起伏**：
```
✓ 好的曲线：▃▅▆█▅▄▆▇█▆（有波峰有波谷）
✗ 坏的曲线：▄▄▄▄▄▄▄▄▄▄（平的）
```

**2. 有递进**：
```
✓ 好的曲线：▃▅▆█（逐步攀升）
✗ 坏的曲线：█▆▅▃（逐步下降）
```

**3. 有喘息**：
```
✓ 好的曲线：▆▇█▇▆→▃▄（高潮后有缓冲）
✗ 坏的曲线：▇█▇█▇█▇█（连续高潮无喘息）
```

### B. 不同阶段的节奏特征

| 阶段 | 平均张力 | 高潮密度 | 特征 |
|------|---------|---------|------|
| 前期（卷1-3） | 4.0-4.5 | 低-中 | 建立期待，稳步升温 |
| 中期（卷4-7） | 4.5-5.0 | 中-高 | 持续精彩，偶有波动 |
| 后期（卷8-10） | 5.0-6.0 | 高-极高 | 高潮迭起，冲向终局 |

### C. 常见节奏问题诊断

**问题1：虎头蛇尾**
- **症状**：前3卷平均张力5.0，后3卷平均张力4.0
- **原因**：前期投入太多，后期创意枯竭/疲劳
- **解决**：重新分配资源，后期增加爆点和冲突

**问题2：中期拖沓**
- **症状**：卷4-7平均张力低于卷1-3
- **原因**：中期承上启下，缺少明确目标
- **解决**：强化"中点"设计，增加中期高潮

**问题3：开局过猛**
- **症状**：卷1平均张力6.0，后续卷无法超越
- **解决**：降低卷1张力，留出递进空间

**问题4：过山车**
- **症状**：相邻卷张力剧烈波动（如5.0→3.0→5.5）
- **原因**：缺少整体规划
- **解决**：平滑过渡，避免断崖式下降

## 常见问题

**Q1：我的全书平均张力只有4.2，是不是太低了？**
A：不一定。关键看：
- 是否有起伏？（不是平的）
- 是否递进？（后期>前期）
- 关键节点张力是否够？（中点、失去一切、决战≥6级）

如果满足以上条件，4.2也可以。网文不需要"全程高潮"，适度起伏更健康。

**Q2：连续2卷平均张力下降怎么办？**
A：允许单卷波动，但不允许连续下降。建议：
- 检查这两卷是否缺少高潮章节
- 检查是否过渡章节过多
- 增加冲突和爆点密度

**Q3：节奏总监和冲突地图有什么区别？**
A：
- **冲突地图**：单卷级，关注冲突类型和密度
- **节奏总监**：全书级，关注跨卷节奏和递进

两者配合使用：先用冲突地图优化单卷，再用节奏总监检验全书。

**Q4：我的节奏心电图很平，怎么办？**
A：说明缺少起伏。建议：
- 增加高潮章节（≥6级张力）
- 拉大波峰和波谷的差距
- 不要每章都是"中等张力"（单调）

**Q5：每次运行都要重新分析所有卷吗？**
A：是的。因为节奏总监关注"跨卷节奏"，必须综合分析才能发现问题。但可以重点查看新规划的卷与之前卷的衔接。
