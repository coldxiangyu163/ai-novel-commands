# 文笔风格学习：$ARGUMENTS

**从优秀作品中提取可模仿的写作技巧，应用到自己的创作中。**

> **核心理念**：
> - 学习文笔不是抄袭，是理解"为什么这样写更好"
> - 好的文笔是可拆解、可学习、可应用的
> - 目标是形成自己的风格，而非复制别人

```

        ↓
/learn-style          → 文笔风格学习 → 直接生成 写作风格.md

```

**注意**：`/learn-style` 已替代 `/design-style` 的功能，如果执行了 `/learn-style`，不需要再执行 `/design-style`。

## 使用建议

1. **选择合适的对标**：文笔风格要与本书定位匹配
2. **深度分析几章比泛泛看全书有效**：选2-3个精彩章节深入拆解
3. **提炼可操作的要点**：不是"写得好"，而是"怎么写好"
4. **及时应用**：学完马上写一章练习
5. **形成自己的风格**：模仿是手段，创新是目的
