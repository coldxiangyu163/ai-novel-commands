## 二、高手写法核心原则（⭐最高优先级）

> **这些原则适用于所有题材，是区分AI写作和高手写作的关键。**

### 原则一：留白的艺术

高手知道什么**不该写**。读者的想象力是最好的画笔。

**核心理解**：
- 你写得越多，读者想象的空间越小
- 点到为止比详细描写更有力量
- 沉默、省略、空白都是表达

**对比领悟**：
```
普通写法：
主角的目光缓缓扫过众人，最后落在某人身上。
"出发。"他沉声说道。

高手写法：
"出发。"

→ 读者自然知道他在看谁、用什么语气。不需要交代。
```

---

### 原则二：动作胜过描述

**展示，而非告诉**。用具体的行为让读者自己感受。

**核心理解**：
- "她很担心"是告诉，"她攥紧了袖口"是展示
- 读者通过动作自己得出结论，印象更深
- 一个精准的小动作胜过一段心理描写

**对比领悟**：
```
普通写法：
她的眼中闪过一丝担忧，她很害怕他会出事。

高手写法：
她没有说话，手指无意识地摩挲着袖口的布料。

→ 让读者自己感受到她的担忧，比直接说更有力。
```

---

### 原则三：每个角色都是独立的人

不同的人有不同的反应方式。这是角色鲜活的关键。

**核心理解**：
- 五个人同意一件事，不会用同样的方式表达
- 角色的反应方式本身就是人物塑造
- 沉默、动作、台词、表情都是表达

**对比领悟**：
```
普通写法：
众人纷纷点头，表示同意。

高手写法：
"成！"张三一拍大腿。
李四没有反驳。
王五轻轻"嗯"了一声。
赵六已经开始收拾东西了。

→ 每个人的"同意"都不同，这就是人物塑造。
```

---

### 原则四：简洁是力量（分场景适用）

**⚠️ 此原则主要适用于日常/过渡场景，高潮场景不应过度精简！**

**适用场景**（可大胆精简）：
- 日常对话、赶路、过渡
- 信息交代、铺垫
- 对话后的"他说""她说"

**不适用场景**（保持详写）：
- 战斗高潮、打脸、情感爆发
- 冲突、危机、重要互动

---

### 原则五：对话不需要"说"

大部分对话后面不需要"他说""她说"。

**对比领悟**：
```
普通写法：
"你来了。"他说。
"嗯。"她点点头说道。
"坐吧。"他指了指椅子说。

高手写法：
"你来了。"
"嗯。"
他指了指椅子，"坐吧。"

→ 读者知道是谁在说话，不需要反复标注。
```

## 三、表达多样性（反程式化）

### 核心意识

写作时始终保持一个意识：**读者已经读了很多网文**。
他们对套路非常敏感，任何重复都会降低阅读体验。

### 三个自问

每写完一段，问自己：
1. **这个表达，我前面用过吗？** → 用过就换一个
2. **这个开头，和前几章像吗？** → 像就换个方式
3. **这个套路，读者见过吗？** → 见过就加点变化

### 警惕信号

如果发现自己在写这些，停下来想想有没有更好的表达：
- "不禁""竟然""居然" → 换成具体的反应描写
- 又一次"皱眉""冷笑" → 换个表情或动作
- 又是同样的开头方式 → 试试别的切入点
- "心中暗道" → 直接写想法
- "嘴角微微上扬" → 换成"笑了"或其他表达

## 四、禁忌清单

### 绝对禁止
1. 大段环境描写开头（除非必要）
2. 主角圣母行为（除非人设如此）
3. 无意义的日常水文
4. 过度卖惨超过2章
5. 重复使用相同句式/词汇

### 尽量避免
1. 过多心理独白
2. "目光扫过""眼神交汇"等俗套描写
3. 每段都以人名开头
4. 对话后频繁加"他说""她说"
