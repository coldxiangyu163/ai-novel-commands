# 创意验证

**在投入大量时间设计世界观和人物之前，先验证你的创意是否有市场潜力。**

> **核心理念**：
> - 再好的执行也救不了一个无人问津的创意
> - 先验证方向，再精雕细节
> - 避免写出"自嗨"的作品

---

## 第一步：读取项目信息

读取以下文件：
1. `AGENTS.md` - 获取书名、题材、卖点
2. `小说规划.md` - 获取核心设定

---

## 第二步：一句话测试

> **如果你不能用一句话让人想看，那你的书可能真的不够吸引人。**

### 2.1 提炼核心卖点

**一句话公式**：
```
[身份/处境反差] + [金手指/能力] + [核心目标] + [核心冲突/悬念]
```

**模板示例**：
- 末世重生："十年末世老兵重生末世前三天，带着仇恨和系统，要让所有害他的人付出代价"
- 玄幻逆袭："被家族抛弃的废物少年，觉醒上古神兽血脉，一步步碾压曾经欺辱他的人"
- 都市神豪："穷小子意外获得亿万富翁系统，开始疯狂打脸所有看不起他的人"

**请提炼你的一句话**：
```markdown
## 一句话卖点

【核心一句话】：
_________________________________________________

【分解检验】：
- 身份反差：[有没有？是什么？]
- 金手指：[是什么？有吸引力吗？]
- 核心目标：[主角要做什么？]
- 核心悬念：[读者想知道什么？]
```

### 2.2 三秒吸引力测试

| 测试项 | 是/否 | 说明 |
|--------|-------|------|
| 听完一句话，会想问"然后呢？" | | |
| 有让人好奇的悬念或反转 | | |
| 主角的目标清晰且有吸引力 | | |
| 有"如果是我会怎么做"的代入空间 | | |

**如果4项中有2项以下是"是"，建议重新打磨核心创意。**

### 3.2 差异化检验

```markdown
## 差异化分析

【你的独特之处】：
- 设定差异：[你有什么同类书没有的设定？]
- 卖点差异：[你的核心爽点有什么不同？]

【差异化风险评估】：
```

## 第四步：读者画像

> **你为谁写？他们想要什么？**

### 4.1 目标读者定义

```markdown
## 读者画像

### 基本特征
- 年龄段：XX-XX岁
- 性别：男性为主 / 女性为主 / 均衡
- 阅读平台：起点 / 番茄 / 七猫 / 其他
- 阅读时间：通勤 / 睡前 / 碎片时间

### 阅读偏好
- 核心诉求：爽 / 代入 / 甜 / 虐 / 学习
- 喜欢的节奏：快节奏 / 慢热 / 起伏型
- 能接受的篇幅：长篇 / 中篇 / 短篇
- 付费意愿：高 / 中 / 低

### 心理需求
- 现实中缺什么：[他们在书里找什么补偿？]
- 核心幻想：[他们想成为什么样的人？]
- 情绪出口：[他们需要释放什么情绪？]
```

### 4.2 读者期待匹配

| 读者期待 | 你的作品是否满足 | 具体设计 |
|----------|------------------|----------|
| 强代入感 | 是/否 | [怎么满足？] |
| 持续爽感 | 是/否 | [怎么满足？] |
| 情感满足 | 是/否 | [怎么满足？] |
| 期待感 | 是/否 | [怎么满足？] |

## 第六步：书名与简介设计

> **书名和简介决定读者是否点进来**

### 6.1 书名设计

**书名三要素**：
1. **类型信号**：让读者一眼知道是什么类型
2. **差异化**：与同类书名有区别
3. **记忆点**：容易记住、容易搜索

**书名检验**：
| 检验项 | 状态 |
|--------|------|
| 能看出是什么类型？ | |
| 与同类书名有区别？ | |
| 好记、好搜索？ | |
| 有一点悬念/好奇感？ | |

### 6.2 简介设计

**简介公式**：
```
[身份设定] + [核心能力/金手指] + [核心目标] + [核心冲突] + [情绪钩子]
```

**简介模板**：
```markdown
[第一段：身份+处境]
唐健，末世十年的幸存者，五阶觉醒者，在踏入巅峰的最后一步被围杀而死。

[第二段：转折+金手指]
再次睁眼，他回到了末世爆发前三天。这一次，他有了系统，有了前世的记忆。

[第三段：目标+冲突]
前世害他的人，一个都跑不掉。前世没能保护的人，这一世死也要护住。

[第四段：情绪钩子/悬念]
当末世再次降临，这一次，他将站在所有人的尸骨之上，俯瞰这个重生的世界。
```

**简介检验**：
| 检验项 | 状态 |
|--------|------|
| 前两句能抓住注意力？ | |
| 金手指/特色清晰？ | |
| 目标明确？ | |
| 有想看下去的冲动？ | |


---

## 注意事项

1. **诚实评估**：不要自我安慰，市场不会说谎
2. **可以迭代**：验证不通过可以调整后重新验证
3. **及时止损**：如果创意确实有硬伤，早发现比写10万字后发现好
4. **保持开放**：好创意往往在迭代中诞生
