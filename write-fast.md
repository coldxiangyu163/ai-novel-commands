# 快速写作：$ARGUMENTS

> **核心理念**：风格第一，流程从简。写出来的东西要像网文，不像AI。

---

## 第一步：读取（只读必要文件）

1. `AGENTS.md` — 核心设定、字数要求
2. `卷X大纲.md` — 提取本章剧情点
3. `前2章正文` — 保持衔接
4. `人物档案` — 根据大纲对应章节按需读取
5. `世界观` — 根据大纲对应章节按需读取

---

## 第二步：风格铁律（⭐写作全程贯彻）

**写作时必须对照 `写作风格.md` 整体内容进行**

---

## 第三步：输出

1. **章节正文**（直接输出完整内容）

2. **保存文件**
   - 创建：`卷X-卷名/章节/第XX章-标题.md`
   - 更新：`卷X-卷名/章节摘要.md`（追加一行摘要）

3. **简要确认**
```
✓ 第X章完成，XXXX字
✓ 已保存
```

## 对比示例

### ❌ AI味写法
```
沈砚清站在二楼的缓步台上，居高临下地俯视着楼下那个刚刚推门而入的男人。她的嘴角微微上扬，勾起一抹意味深长的弧度，眼底闪烁着复杂难明的光芒，仿佛一只正在审视猎物的优雅猎豹。

她轻轻晃动手中那只晶莹剔透的红酒杯，殷红的酒液在杯中缓缓旋转，映照着她精致绝美的侧脸。那一袭酒红色的真丝睡裙在昏黄的灯光下泛着柔和的光泽，衬得她整个人都散发着一种慵懒而危险的气息。
```

### ✓ 网文写法
```
二楼。

沈砚清端着红酒杯，看着楼下那个男人。

嘴角一勾。

"宋总，你是打算在那儿站成石像，还是上来领你的结婚周年礼物？"

宋砚愣住了。

这女人……今天吃错药了？
```

## 注意

- **不要追求文采**，追求爽感
- **不要长篇大论**，要短平快
- **不要面面俱到**，抓重点
- 写不下去就跳过，先把剧情点写完
- 风格比流程重要
