# 三维冲突地图：$ARGUMENTS

**在 plan-volume 之后使用，检验卷大纲的冲突密度和层次。**

> **核心理念**：
> - 好故事的本质是冲突，没有冲突就是流水账
> - 冲突不是只有"打架"，还有人际冲突和内心冲突
> - 三维冲突叠加的章节，爆发力是单维冲突的3倍

---

## 第一步：理解三维冲突模型

### 1.1 什么是三维冲突？

**三维冲突模型**将故事冲突分为三个层次：

```
冲突金字塔：

              内心冲突（人vs自己）
              ├─ 价值观冲突
              ├─ 两难抉择
              └─ 恐惧克服
                    ▲
              人际冲突（人vs人际关系）
              ├─ 理念分歧
              ├─ 信任危机
              └─ 利益冲突
                    ▲
         外部冲突（人vs人/环境）
         ├─ 敌人追杀
         ├─ 资源争夺
         └─ 生存危机
```

### 1.2 为什么需要三维冲突？

| 问题 | 三维冲突如何解决 |
|------|-----------------|
| 只有打斗场景，读者觉得单调 | 加入人际冲突（队友分歧）和内心冲突（主角挣扎） |
| 主角太顺，缺少张力 | 制造人际冲突（盟友质疑）或内心冲突（道德困境） |
| 高潮章节不够爆 | 三维冲突叠加：外部敌人+队友撤退+主角内心挣扎 |

### 1.3 三维冲突示例

**单维冲突场景**（张力★★☆☆☆）：
> 主角被仇人追杀，逃跑。

**三维冲突场景**（张力★★★★★）：
> 主角被仇人追杀（外部冲突），队友因为害怕要撤退（人际冲突），主角内心挣扎：是牺牲队友独自逃生，还是为了队友冒险反击？（内心冲突）

**效果对比**：
- 单维：读者只关心"能不能逃掉"
- 三维：读者同时关心"能不能逃掉"+"队友会不会散"+"主角会怎么选"

### 3.2 冲突强度判定标准

| 强度 | 外部冲突 | 人际冲突 | 内心冲突 |
|------|----------|----------|----------|
| ★☆☆☆☆ | 小摩擦，轻松化解 | 小分歧，一句话解决 | 微弱犹豫，瞬间决定 |
| ★★☆☆☆ | 小冲突，有危险但可控 | 分歧，需要沟通 | 短暂挣扎，较快决定 |
| ★★★☆☆ | 中等冲突，有受伤风险 | 矛盾，关系紧张 | 明显挣扎，艰难决定 |
| ★★★★☆ | 重大冲突，生死危机 | 严重矛盾，关系破裂边缘 | 深刻挣扎，痛苦抉择 |
| ★★★★★ | 终极对决，不是你死就是我亡 | 关系彻底决裂/背叛 | 灵魂拷问，撕裂性选择 |

### 3.3 冲突层次判定

- **单维冲突**：只有一种冲突（如只有外部冲突）
- **二维冲突**：同时有两种冲突（如外部+人际，或外部+内心）
- **三维冲突**：三种冲突同时存在并交织

**冲突层次要求**：
- 平常章节：至少单维，建议二维
- 重要章节：建议二维，力争三维
- 高潮章节：必须三维

## 注意事项

1. **冲突不等于打斗**：人际冲突和内心冲突同样重要，甚至更有深度
2. **适度原则**：不是每章都要三维冲突，过渡章节可以是单维
3. **冲突要真实**：冲突必须有合理的起因和后果，不能为了冲突而冲突
4. **冲突要递进**：后期的冲突强度应该高于前期
5. **与感情线协同**：感情线的冲突也可以是人际冲突（如误会）或内心冲突（如心动vs理智）

## 附录：三维冲突典型场景设计

### A. 三维冲突叠加示例

**场景：主角团队遭遇强敌**

**单维冲突版**（张力★★★☆☆）：
> 主角团队遭遇强敌，经过苦战，最终击败敌人。

**三维冲突版**（张力★★★★★）：
> 主角团队遭遇强敌（外部冲突）。战斗中，副队长因害怕想带人撤退，与主角发生激烈争吵（人际冲突）。主角内心挣扎：是坚持理想继续战斗可能全员阵亡，还是听从副队长撤退但放弃任务？他想起女主对他说过的话"有些事，比活着更重要"，最终决定战斗（内心冲突+主角成长）。副队长见状，虽然恐惧但选择留下。最终团队齐心协力，险胜（人际冲突化解）。

**效果对比**：
- 单维：只是一场战斗
- 三维：读者经历了3层情绪波动，印象深刻

### B. 不同题材的三维冲突应用

**末世生存文**：
- 外部冲突：丧尸/敌对势力/资源短缺
- 人际冲突：团队内部分配矛盾/信任危机
- 内心冲突：人性vs兽性/道德vs生存

**都市重生文**：
- 外部冲突：商业竞争/仇人报复
- 人际冲突：与家人的关系修复/与前世爱人的纠葛
- 内心冲突：重生后的身份认同/要不要改变命运

**修仙升级文**：
- 外部冲突：敌对宗门/天劫/魔族
- 人际冲突：门派内部争斗/师徒矛盾
- 内心冲突：修仙vs凡心/正道vs魔道/复仇vs放下

### C. 人际冲突设计技巧

**常见误区**：
- ❌ 为了冲突而冲突（队友无缘无故闹矛盾）
- ❌ 冲突太容易化解（一句话就和好）
- ❌ 冲突人物脸谱化（提意见的都是反派）

**正确做法**：
- ✓ 冲突有合理起因（利益/理念/误会）
- ✓ 冲突有发展过程（积累→爆发→化解/激化）
- ✓ 双方都有道理（灰度冲突，不是黑白对立）

**示例**：
> 主角要拯救一个可能已被感染的幸存者，副队长坚决反对（理念分歧）。副队长的理由有道理（带回感染者可能害死全队），主角的理由也有道理（不能见死不救）。这是灰度冲突，没有标准答案，更有深度。

### D. 内心冲突设计技巧

**常见误区**：
- ❌ 主角从不犹豫（太完美）
- ❌ 犹豫时间太短（一段内心独白就决定了）
- ❌ 选择没有代价（选哪个都无所谓）

**正确做法**：
- ✓ 主角有弱点/恐惧/价值观冲突
- ✓ 犹豫有合理时长（重大抉择需要时间）
- ✓ 选择有代价（每个选择都要付出）

**两难抉择设计模板**：
1. **情境**：主角必须在X和Y中选一个
2. **选项A**：救女主，但任务失败，很多人会死
3. **选项B**：完成任务救多数人，但女主会死
4. **无完美答案**：无论选哪个都有重大代价
5. **主角选择**：体现主角的价值观/成长

## 常见问题

**Q1：我的每章都有外部冲突，为什么冲突地图说"单一化风险"？**
A：冲突类型单一。如果80%都是"战斗对抗"，虽然每章有冲突，但读者会觉得重复。建议增加资源争夺、环境威胁、时间压力等其他类型。

**Q2：人际冲突会不会让主角显得软弱？**
A：恰恰相反。敢于直面人际冲突的主角更真实、更有魅力。队友提意见不代表主角软弱，如何处理分歧才体现主角的领导力和成长。

**Q3：内心冲突会不会拖慢节奏？**
A：适度的内心冲突不会拖慢节奏，反而增加深度。关键是：
- 不要过度渲染（2-3段内心戏即可）
- 与行动结合（边行动边挣扎，不要停下来长篇独白）
- 在高潮章节使用（平常章节可以简化）

**Q4：三维冲突叠加会不会太复杂？**
A：只在高潮章节使用三维冲突叠加，平常章节单维或二维即可。且三维冲突不是"三个独立冲突"，而是"三个冲突交织"：
- ❌ 先打架（外部），打完吵架（人际），吵完独白（内心）← 割裂
- ✓ 打架时队友要撤（人际），主角边打边挣扎（内心），三者同时发生 ← 融合
