# 人物内核深度设计：$ARGUMENTS

**好故事的核心是人物转变。本命令设计人物的"灵魂级"内核，让角色弧光直击人心。**

> **核心理念**：
> - 表面目标（Want）让读者追看，内在需求（Need）让读者感动
> - 人物弧光的本质：放弃错误信念（Lie），接受故事教给他的真相（Truth）
> - 创伤（Wound）是一切的根源，它塑造了角色的一切

---

## 第二步：理解人物内核五要素

### 2.1 内核五要素模型

```
                    ┌─────────────────┐
                    │   Wound（创伤） │  ← 过去发生了什么？
                    │    一切的根源   │
                    └────────┬────────┘
                             │
                             ▼ 导致
                    ┌─────────────────┐
                    │   Lie（谎言）   │  ← 他因此相信什么错误的东西？
                    │   错误的信念    │
                    └────────┬────────┘
                             │
              ┌──────────────┼──────────────┐
              │              │              │
              ▼              ▼              ▼
      ┌───────────┐  ┌───────────┐  ┌───────────┐
      │Want（欲望）│  │Need（需求）│  │Ghost（幽灵）│
      │ 表面想要的 │  │ 真正需要的 │  │ 行为模式  │
      └───────────┘  └───────────┘  └───────────┘
              │              │              │
              └──────────────┼──────────────┘
                             │
                             ▼ 故事中
                    ┌─────────────────┐
                    │  Truth（真相）  │  ← 他必须学会什么？
                    │   正确的信念    │
                    └─────────────────┘
```

### 2.2 五要素定义

| 要素 | 定义 | 问题 | 示例 |
|------|------|------|------|
| **Wound（创伤）** | 过去发生的痛苦事件 | 什么经历伤害了他？ | 被最信任的人背叛 |
| **Lie（谎言）** | 因创伤而形成的错误信念 | 他因此相信什么？ | "我不需要任何人" |
| **Want（欲望）** | 表面上追求的目标 | 他说他想要什么？ | 复仇、变强、生存 |
| **Need（需求）** | 内心真正需要的东西 | 他真正需要什么？ | 被信任、被爱、找到归属 |
| **Ghost（幽灵）** | 创伤导致的行为模式 | 他因此如何行动？ | 拒绝团队、保持距离 |
| **Truth（真相）** | 故事教给他的正确信念 | 他必须学会什么？ | "信任值得冒险" |

## 第三步：挖掘角色创伤（Wound）

> **创伤是一切的根源。没有创伤，就没有谎言；没有谎言，就没有弧光。**

### 3.1 创伤类型速查

| 创伤类型 | 描述 | 导致的谎言 | 典型表现 |
|----------|------|-----------|---------|
| **背叛型** | 被信任的人出卖 | "不能相信任何人" | 孤僻、多疑、测试他人 |
| **抛弃型** | 被重要的人抛弃 | "我不值得被爱" | 先抛弃别人、拒绝亲密 |
| **失败型** | 重大的失败经历 | "我不够好" | 过度努力、害怕尝试 |
| **无力型** | 无法保护重要的人 | "只有力量才能保护" | 追求极端力量 |
| **欺骗型** | 发现世界的真相是谎言 | "一切都是假的" | 愤世嫉俗、难以信任 |
| **羞耻型** | 被羞辱或嘲笑 | "我必须证明自己" | 争强好胜、敏感自尊 |
| **控制型** | 被过度控制/剥夺自由 | "我必须掌控一切" | 控制欲强、难以放手 |
| **忽视型** | 长期被忽视 | "我不重要" | 渴望关注、讨好型 |

## 第四步：提炼角色谎言（Lie）

> **谎言是角色相信的错误信念。它是创伤的产物，也是角色弧光的起点。**

### 4.1 谎言公式

**公式**：因为[创伤]，所以我相信[谎言]

**示例**：
- 因为「被最信任的人背叛」，所以我相信「不能相信任何人」
- 因为「无法保护重要的人」，所以我相信「只有绝对的力量才能保护」
- 因为「被抛弃」，所以我相信「我不值得被爱」

### 4.2 常见谎言清单

| 类型 | 谎言 | 对应创伤 | 行为表现 |
|------|------|----------|----------|
| **关于他人** | "不能相信任何人" | 背叛 | 多疑、孤僻 |
| **关于他人** | "人都是自私的" | 被利用 | 先发制人、功利 |
| **关于自我** | "我不值得被爱" | 抛弃 | 拒绝亲密、自我破坏 |
| **关于自我** | "我不够好" | 失败 | 过度努力、害怕失败 |
| **关于自我** | "我必须独自承担" | 忽视 | 不求助、扛所有 |
| **关于世界** | "弱肉强食是法则" | 无力 | 追求力量、碾压弱者 |
| **关于世界** | "感情是弱点" | 失去 | 隔离情感、避免依赖 |
| **关于成功** | "只有强大才有价值" | 羞辱 | 争强好胜 |
| **关于关系** | "先离开比被抛弃好" | 抛弃 | 先推开别人 |

## 第五步：区分Want与Need

> **Want是角色追逐的目标，Need是角色真正需要的东西。两者的冲突是故事的精髓。**

### 5.1 Want vs Need 对比

| 维度 | Want（外在欲望） | Need（内在需求） |
|------|------------------|------------------|
| **来源** | 有意识的目标 | 无意识的渴望 |
| **驱动力** | 谎言驱动 | 真相驱动 |
| **可见性** | 角色清楚知道 | 角色可能不自知 |
| **对外部** | 改变外部世界 | 改变内在自我 |
| **故事作用** | 推动情节发展 | 推动人物成长 |
| **满足感** | 短暂/空虚 | 真正的满足 |

### 5.2 经典Want vs Need示例

| 作品 | Want | Need | 冲突 |
|------|------|------|------|
| 《钢铁侠》 | 保护世界 | 学会依靠团队 | 自己扛vs信任队友 |
| 《冰雪奇缘》 | 控制魔法 | 接受真实的自己 | 隐藏vs展示 |
| 《斗破苍穹》 | 变强复仇 | 找到自我价值 | 力量vs内心 |
| 《诡秘之主》 | 回到地球 | 找到真正的归属 | 离开vs留下 |

### 5.3 设计Want与Need

```markdown
## [角色名] 的Want与Need

### Want（外在欲望）
**主角说他想要**：
- 具体目标：[如：杀死所有仇人/变成最强/拯救世界]
- 为什么想要：[表面原因]
- 认为得到后会：[角色以为会怎样]

### Need（内在需求）
**主角真正需要**：
- 真正需求：[如：被信任/被爱/找到归属/原谅自己]
- 为什么需要：[这才能治愈创伤]
- 得到后会：[真正的满足/完整]

### Want与Need的关系
| 关系类型 | 本作设计 |
|----------|----------|
| 互斥型 | 追求Want会远离Need？ |
| 并行型 | Want和Need可以同时追求？ |
| 递进型 | 先满足Want才能意识到Need？ |
| 转化型 | 最终Want转化为Need？ |

### 核心张力
**如果只追Want不追Need**：
- 结果：[即使得到Want也会空虚/失去更重要的东西]

**如果放弃Want追求Need**：
- 结果：[可能失去Want的目标，但获得真正的满足]

**本书的处理**：
- [主角是否需要在Want和Need之间选择？]
- [还是找到两者兼得的第三条路？]
```

## 第七步：设计Truth（真相）

> **Truth是故事教给角色的正确信念。它是谎言的对立面，是角色弧光的终点。**

### 7.1 Truth的设计原则

**Truth必须满足**：
- 与Lie直接对立（是Lie的反面）
- 能治愈Wound（解决根本问题）
- 满足Need（给角色真正需要的）
- 服务主题（体现全书主题）

### 7.2 Lie → Truth 转化示例

| Lie（谎言） | Truth（真相） |
|-------------|---------------|
| "不能相信任何人" | "信任值得冒险" |
| "我不值得被爱" | "我值得被爱，也可以爱人" |
| "只有力量才能保护" | "真正的力量是有人愿意为你而战" |
| "感情是弱点" | "感情是力量的源泉" |
| "先离开比被抛弃好" | "值得冒被伤害的风险去建立连接" |
| "我必须独自承担" | "接受帮助不是软弱" |

### 阶段三：Lie与Truth的拉锯（卷X~卷X）
- **动摇时刻**：[什么时候开始动摇？]

### 8.2 弧光节点与节拍表对应

## 注意事项

1. **创伤要具体**：不能是抽象的"受过伤"，必须是具体事件
2. **谎言要可信**：角色相信Lie必须有合理原因
3. **Truth要难得**：接受Truth必须付出代价，不能轻易获得
4. **弧光要渐进**：不能一夜之间改变，需要经历挣扎
5. **行为要一致**：Lie和Ghost必须体现在日常行为中
6. **服务主题**：角色弧光是主题论证的载体
