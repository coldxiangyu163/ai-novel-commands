# 创意激发器：$ARGUMENTS

**目标：打破"第一反应"，找到让读者"没想到"的设计。**

> **核心理念**：
> - 你的第一个想法，读者也能猜到
> - 出彩 = 意料之外 + 情理之中
> - 不是为了反转而反转，是为了更好的故事

---

## 第一步：锁定目标场景

### 1.1 解析输入

从 `$ARGUMENTS` 提取：
- **场景/节点描述**：要激发创意的具体内容
- **卷号/章号**（可选）：如提供，读取上下文

### 1.2 读取上下文（如有卷号章号）

| 文件 | 提取内容 |
|------|----------|
| `卷X大纲.md` | 前后章节、当前设计 |
| `剧情设计.md` | 主线走向、核心冲突 |
| `人物档案/` | 相关人物的核心设定 |
| `全书记忆/已用套路库.md` | 避免重复 |

### 1.3 明确创意目标

```markdown
## 创意目标

**当前场景**：[用户输入的场景描述]
**当前设计**：[如有，从大纲提取]
**创意需求**：
- [ ] 需要更出彩的开场
- [ ] 需要更震撼的高潮
- [ ] 需要更意外的反转
- [ ] 需要更深刻的情感
- [ ] 需要更独特的设定
- [ ] 其他：___
```

## 第二步：预期分析（找出"俗套"）

### 2.1 读者预期清单

**强制列出读者最可能猜到的5个走向：**

```markdown
## 读者预期清单

针对场景「[场景描述]」，读者最可能预期：

| # | 预期走向 | 为什么俗套 | 俗套程度 |
|---|----------|-----------|---------|
| 1 | [最直接的发展] | [在XX作品中常见] | ★★★★★ |
| 2 | [次直接的发展] | [类型套路] | ★★★★☆ |
| 3 | [常见变体A] | [变体但仍可预测] | ★★★☆☆ |
| 4 | [常见变体B] | [稍有新意但不够] | ★★☆☆☆ |
| 5 | [你的第一反应] | [诚实评估] | ★★★?☆ |
```

### 2.2 套路来源分析

```markdown
## 套路来源

这个场景的"标准写法"来自：
- **同类型爆款**：[列出2-3部，它们怎么写的]
- **类型惯例**：[这个类型的标准处理方式]
- **你的习惯**：[你之前类似场景怎么写的]
```

## 第三步：六维创意激发（核心）

### 3.1 逆向思考法

**把每个预期完全颠倒：**

```markdown
## 逆向思考

| 预期 | 逆向 | 可行性 | 如何合理化 |
|------|------|--------|-----------|
| 主角赢 | 主角输 | ?/5 | [需要什么铺垫] |
| 敌人是坏人 | 敌人有苦衷/是受害者 | ?/5 | [如何揭示] |
| 队友帮忙 | 队友背叛/无法帮忙 | ?/5 | [动机是什么] |
| 危机解除 | 危机升级/转化 | ?/5 | [升级到什么] |
| 获得奖励 | 奖励是陷阱/代价 | ?/5 | [代价是什么] |

**最佳逆向方案**：[选出可行性最高且最有冲击力的]
```

### 3.2 视角切换法

**从不同角色视角重新设计这个场景：**

```markdown
## 视角切换

### 从反派视角
- 反派此刻在想什么？
- 反派的计划是什么？
- 如果反派是主角，这个场景怎么写？
- **新发现**：[从反派视角发现的新可能]

### 从配角视角
- 配角此刻在想什么？
- 配角有什么主角不知道的信息？
- 如果配角是主角，会怎么做？
- **新发现**：[从配角视角发现的新可能]

### 从旁观者视角
- 一个路人看到这个场景会怎么想？
- 这个场景在"历史"中会被怎么记载？
- **新发现**：[从旁观者视角发现的新可能]

### 从读者视角
- 读者此刻最想看到什么？
- 读者最怕看到什么？
- 读者会在哪里放下手机？
- **新发现**：[从读者视角发现的新可能]
```

### 3.3 极端假设法

**把情况推到极端，看看会发生什么：**

```markdown
## 极端假设

### 如果主角此刻死了
- 故事会怎么发展？
- 谁会接替主角？
- 这个"死亡"能不能是假的/暂时的？
- **启发**：[从这个假设得到的创意]

### 如果主角此刻黑化
- 主角会做什么？
- 其他人会怎么反应？
- 这个"黑化"能不能是伪装/误解？
- **启发**：[从这个假设得到的创意]

### 如果主角此刻失去一切
- 失去什么最痛？
- 失去后会怎么重建？
- 这个"失去"能不能是成长的代价？
- **启发**：[从这个假设得到的创意]

### 如果时间倒流/快进
- 倒流到什么时候能改变结果？
- 快进10年这个场景会被怎么回忆？
- **启发**：[从时间维度得到的创意]
```

### 3.4 跨界嫁接法

**从其他题材/领域借鉴创意：**

```markdown
## 跨界嫁接

### 如果这是一部悬疑片
- 这个场景隐藏了什么真相？
- 谁在说谎？
- 什么证据被忽略了？
- **嫁接方案**：[悬疑元素如何融入]

### 如果这是一部喜剧
- 这个场景最荒诞的展开是什么？
- 有什么误会可以制造？
- 有什么反差可以利用？
- **嫁接方案**：[喜剧元素如何融入]

### 如果这是一部文艺片
- 这个场景的象征意义是什么？
- 有什么隐喻可以加入？
- 情感的最深层是什么？
- **嫁接方案**：[文艺元素如何融入]

### 如果这是真实事件
- 真实世界中类似情况怎么发展？
- 有什么反常识的真实案例？
- 人性在极端情况下的真实反应？
- **嫁接方案**：[真实感如何增强]
```

### 3.5 元素重组法

**打乱现有元素，重新组合：**

```markdown
## 元素重组

### 当前元素清单
- 人物：[A、B、C...]
- 地点：[X]
- 时间：[Y]
- 冲突：[Z]
- 道具：[...]
- 情感：[...]

### 重组方案

| 重组 | 新组合 | 效果 |
|------|--------|------|
| 换人物 | 如果是[B]而不是[A]来做这件事 | [会怎样] |
| 换地点 | 如果发生在[新地点] | [会怎样] |
| 换时间 | 如果发生在[更早/更晚] | [会怎样] |
| 加道具 | 如果加入[某个道具] | [会怎样] |
| 加人物 | 如果[某人]突然出现 | [会怎样] |
| 减元素 | 如果去掉[某元素] | [会怎样] |

**最佳重组方案**：[选出最有新意的组合]
```

### 3.6 情感深挖法

**挖掘场景的情感深度：**

```markdown
## 情感深挖

### 表层情感
- 这个场景表面上是什么情感？[愤怒/悲伤/喜悦/恐惧...]
- 读者的直接情感反应是什么？

### 深层情感
- 主角真正害怕的是什么？
- 主角真正渴望的是什么？
- 这个场景触及了主角的什么创伤（Wound）？
- 这个场景挑战了主角的什么信念（Lie）？

### 情感反转
- 能不能从[A情感]转到[B情感]？
- 能不能让读者先笑后哭/先哭后笑？
- 能不能让读者的情感预期落空然后给更强的情感？

### 情感共鸣点
- 这个场景能触及读者的什么经历？
- 有什么普世的情感可以放大？
- **情感设计**：[如何让读者"感同身受"]
```

1. 如果主角此刻做出相反的选择？
2. 如果这个场景发生在10年前/10年后？
3. 如果反派此刻说的是真话？

## 使用建议

1. **关键节点必用**：中点、高潮、卷末等关键节点
2. **卡文时用**：想不出怎么写时
3. **觉得平淡时用**：写完发现太平淡时
4. **不要过度使用**：不是每个场景都需要反转，日常章节可以正常写

**记住**：出彩的本质是"超出预期"，但前提是"合情合理"。
