## 第三步：节拍表校验与对接

> **核心理念**：
> - 节拍表提供全书级的关键节点定位
> - 卷大纲必须对接节拍表，确保关键节点有足够张力
> - 节拍表是"骨架"，卷大纲是"血肉"

### 3.1 读取节拍表

从 `节拍表.md` 或 `节拍表速查.md` 读取全书节拍表，提取本卷涉及的关键节拍：

```markdown
## 本卷涉及的关键节拍

| 节拍 | 全书位置 | 本卷对应章节 | 要求张力 | 要求爆点类型 |
|------|---------|-------------|---------|-------------|
| [节拍名] | 第X章（全书） | 第X章（本卷） | ≥★★★★☆ | 反转/揭秘/... |
| [节拍名] | 第X章（全书） | 第X章（本卷） | ≥★★★★★ | 打脸+觉醒 |
| ... | ... | ... | ... | ... |
```

**示例**：
- 如果本卷是卷一（第1-40章），涉及的节拍可能有：
  - 开场意象（第1章）
  - 触发事件（第40章全书的10% = 约第40章）
  - 如果本卷跨越25%位置，还包含"第二幕开始"

### 3.2 节拍表强制要求

**关键节拍章节必须满足**：
- 张力等级达标（如"中点"必须≥★★★★☆）
- 有对应类型的爆点（如"失去一切"必须有虐心爆点）
- 有三维冲突叠加（关键节拍章节必须是三维冲突）

**节拍表对接检查清单**：
- [ ] 本卷涉及的关键节拍已识别？
- [ ] 每个关键节拍对应的章节已确定？
- [ ] 关键节拍章节的张力设计达标？
- [ ] 关键节拍章节的冲突层次达标（三维）？

## 第四步：全卷节奏预览（⭐先画曲线，再填章节）

> **核心原则**：不要直接跳到章节大纲！先规划全卷节奏，确保整体有起伏，再填充具体内容。

### 4.1 张力曲线设计

**根据本卷章节数，画出张力曲线：**

```
【示例：40章卷】

章节: 1   5   10   15   20   25   30   35   40
张力: ▃   ▅   ▆    ▇    █    ▆    ▇    █    ▇
标注: 开局 升温 触发  复仇 中点  发展 危机 高潮 收尾
```

**张力等级说明**：
| 等级 | 符号 | 说明 | 适用场景 |
|------|------|------|----------|
| 1级 | ▂ | 平静/日常 | 铺垫、过渡 |
| 2级 | ▃ | 轻微波动 | 信息交代、伏笔 |
| 3级 | ▄ | 小冲突 | 小打脸、小危机 |
| 4级 | ▅ | 中等张力 | 升温、蓄力 |
| 5级 | ▆ | 较高张力 | 重要冲突 |
| 6级 | ▇ | 高张力 | 小高潮 |
| 7级 | █ | 最高张力 | 大高潮 |

### 4.2 节奏健康指标与预警

> **核心理念**：
> - 节奏不仅要"有起伏"，还要符合量化健康标准
> - 提前预警拖沓/疲劳风险，避免写作时才发现问题

**量化健康指标**：

| 指标 | 健康值 | 预警阈值 | 说明 |
|------|--------|---------|------|
| 平均张力 | ≥4.0级 | <3.5级 | 过低=拖沓 |
| 张力标准差 | 1.5-2.5 | <1.0或>3.0 | 太平=单调，太剧烈=疲劳 |
| 高潮章节数（≥6级） | ≥总章节×15% | <10% | 高潮太少 |
| 低张力章节数（≤3级） | ≤总章节×20% | >30% | 拖沓章节过多 |
| 最长无高潮间隔 | ≤10章 | >15章 | 拖沓风险 |
| 最长低谷持续 | ≤5章 | >7章 | 拖沓风险 |
| 最长高潮持续 | ≤8章 | >10章 | 疲劳风险 |

**节奏预警系统**：

```markdown
## 节奏预警

### ⚠️ 拖沓风险检测
- **检测规则**：连续5章张力<4级 → 轻度拖沓
- **检测规则**：连续8章张力<4级 → 严重拖沓
- **当前状态**：
  - 第X-X章：连续X章低张力 → [⚠️/✗]
  - 建议：在第X章增加中等冲突或爆点

### ⚠️ 疲劳风险检测
- **检测规则**：连续7章张力≥6级 → 轻度疲劳
- **检测规则**：连续10章张力≥6级 → 严重疲劳
- **当前状态**：
  - 第X-X章：连续X章高张力 → [⚠️/✗]
  - 建议：在第X章增加缓冲章节

### ⚠️ 高潮间隔预警
- **检测规则**：高潮间隔>10章 → 预警
- **当前状态**：
  - 第X章→第X章：间隔X章 → [⚠️/✗]
  - 建议：在第X章增加小高潮
```

### 4.3 关键节点定位

**根据经典叙事结构，锁定关键节点位置：**

| 位置 | 节点名称 | 章节 | 功能 | 内容预设 |
|------|----------|------|------|----------|
| 3-5% | 开篇钩子 | 第1-2章 | 抓住读者 | [必须有吸引力] |
| 10-12% | 触发事件 | 第X章 | 打破日常 | [改变主角处境的事件] |
| 20-25% | 第一高潮 | 第X章 | 第一个大爽点 | [让读者确认追读] |
| 45-55% | 中点 | 第X章 | 转折/反转 | [改变游戏规则] |
| 70-75% | 最低点 | 第X章 | 最大危机 | [主角陷入绝境] |
| 85-90% | 高潮 | 第X章 | 全卷最燃 | [本卷最精彩场面] |
| 95-100% | 收尾钩子 | 最后1-2章 | 引出下卷 | [悬念/新开端] |

**⚠️ 必须先确定这7个节点，再填充中间章节！**

### 6.2 冲突健康指标

**必须达标的指标**：

| 指标 | 健康值 | 说明 |
|------|--------|------|
| 冲突章节占比 | ≥90% | 几乎每章都有冲突 |
| 平均冲突强度 | ≥3.5星 | 不能太平淡 |
| 二维+冲突占比 | ≥50% | 一半以上章节有多维冲突 |
| 三维冲突章节数 | ≥卷章节数×20% | 至少20%章节三维冲突 |
| 高潮章节冲突维度 | 必须3维 | 高潮必须三维冲突叠加 |
| 最长无冲突间隔 | ≤2章 | 不能连续3章无冲突 |
| 最长低张力间隔 | ≤5章 | 不能连续6章低张力 |

**风险预警**：
- ⚠️ 拖沓风险：连续5章张力<3级
- ⚠️ 疲劳风险：连续7章高强度冲突
- ⚠️ 单一化风险：某类冲突占比>80%

### 6.3 根据冲突地图优化大纲

**如果检测到问题，必须调整**：
- 低冲突章节：增加人际冲突或内心冲突
- 高潮章节冲突不足：强化为三维冲突
- 拖沓区域：插入中等冲突或爆点
- 单一化：增加其他类型冲突

    ├─ 绘制张力曲线
    ├─ 量化健康指标检查
    ├─ 节奏预警（拖沓/疲劳）
    ├─ 关键节点定位
    └─ 批次划分
    ↓
第五步：生成卷大纲
    ├─ 卷概述
    ├─ 章节大纲（每章必须标注）★强制：
    │   ├─ 三维冲突
    │   ├─ 弧光阶段（⭐新增）
    │   └─ 表达方式规划（⭐新增）
    ├─ 爽点分布（对接爆点曲线）

    ↓

    ├─ 运行：/design-conflict-map <卷号>

    ├─ 风险预警
    └─ 根据报告优化大纲
    ↓
第七步：约束校验
    ├─ 约束检查（含弧光/爆点/套路/表达约束）

**关键工具调用点**：

4. **每10章**：调用 `/mainline-tracker`（主线进度检查）
5. **每3卷**：调用 `/design-rhythm-master`（全书节奏检验）
